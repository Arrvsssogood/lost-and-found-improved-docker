How It Works

📸

Report an Item

Found an item on campus? Log in and submit it with a photo and location details so it can be returned to its rightful owner.

🔍

Browse the Gallery

Search and filter through all found items. Use the search bar to find yours fast. If an item is labeled "Found", it means someone has reported that they found the item. If it is labeled "Lost", it means the item was reported as missing by its owner. This does not mean it has been claimed or returned yet.

🎉

Get Reunited

Contact the finder or admin to verify ownership and arrange the return of the item. Once the rightful owner successfully claims the item, it is marked as claimed in the system.
